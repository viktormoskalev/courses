Feature control frames apply geometric tolerances to specific features on a part without adding too many notes to a drawing. Figure 36 shows how the feature control frame uses symbols to describe the feature's tolerances. The first part of the frame shows the geometric symbol applied to the feature; the second part of the frame contains the tolerance,

a zone descriptor, and a modifier if necessary; the last part of the frame contains the applicable datum references. Feature control frames are read from left to right and top to bottom if additional frame rows are present.

Some control frames are relatively simple, using only a geometric symbol and a tolerance, while others have additional information to guide print interpretation. Two or more feature control frames can be combined to apply more than one type of geometric tolerance to a feature, as shown in Figure 37. Combined—or composite—control frames can define another datum when a feature must have a defined

orientation to another feature. Further, composite feature control frames can apply a single geometric tolerance to a feature for multiple datums and then tighten the tolerance for one of the datums.
